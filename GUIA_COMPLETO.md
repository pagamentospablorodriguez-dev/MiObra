# Sistema Completo de Gestão de Obras

## O QUE ESTE SISTEMA RESOLVE

Este sistema foi criado especificamente para resolver **TODOS** os problemas do seu tio:

### ✅ Problemas Resolvidos

1. **Não precisa mais estar presente em todas as obras**
   - Monitore tudo remotamente em tempo real
   - Veja quem está trabalhando agora
   - Receba fotos do progresso automaticamente

2. **Funcionários trabalham corretamente mesmo sem supervisão**
   - Sistema de check-in obrigatório com localização
   - Funcionários precisam enviar fotos das tarefas
   - Você aprova ou rejeita cada trabalho com notas
   - Sistema de avaliação para identificar os melhores e piores

3. **Finalmente pode tirar férias**
   - Tudo fica registrado e organizado
   - Alertas automáticos se algo der errado
   - Histórico completo de tudo que acontece

4. **Clientes satisfeitos**
   - Eles veem o progresso em tempo real
   - Recebem atualizações automáticas
   - Podem ver fotos da obra a qualquer momento
   - Menos reclamações e mais confiança

5. **Identifica funcionários ruins rapidamente**
   - Sistema de notas por tarefa
   - Histórico de desempenho de cada um
   - Fácil ver quem trabalha bem e quem não trabalha

6. **Gestão de 4 obras simultaneamente**
   - Dashboard central mostrando TUDO
   - Um lugar só para ver todas as obras
   - Alertas quando algo precisa de atenção

## COMO FUNCIONA

### Para o SEU TIO (Admin)

**Dashboard Principal:**
- Vê TODAS as 4 obras em tempo real
- Sabe quantos funcionários estão trabalhando AGORA
- Vê todos os problemas urgentes
- Aprova ou rejeita trabalhos dos funcionários
- Dá notas de qualidade para cada tarefa
- Acompanha progresso de cada obra
- Recebe notificações de tudo importante

**Aprovação de Tarefas:**
- Funcionário marca tarefa como concluída
- Sistema notifica você
- Você vê as fotos do trabalho
- Dá uma nota de 0-10
- Aprova ou rejeita
- Se rejeitar, diz o que precisa corrigir

**Monitoramento:**
- Vê quem fez check-in hoje
- Quantas horas cada um trabalhou
- Quais tarefas foram concluídas
- Progresso de cada obra em %
- Gastos vs Orçamento

### Para FUNCIONÁRIOS

**Sistema de Presença:**
- Faz check-in ao chegar na obra
- Sistema registra horário e localização
- Durante o dia, atualiza suas tarefas
- Faz check-out ao sair

**Tarefas:**
- Vê suas tarefas pendentes
- Marca como "iniciada" quando começa
- Tira fotos do trabalho
- Envia para revisão quando termina
- Recebe feedback do admin

**Motivação:**
- Sistema de notas motiva a trabalhar bem
- Vê seu próprio histórico
- Recebe notificações de aprovações

### Para CLIENTES

**Portal do Cliente:**
- Vê sua obra em tempo real
- Fotos atualizadas do progresso
- Porcentagem de conclusão
- Lista de tarefas concluídas
- Orçamento e gastos
- Quantos funcionários trabalhando
- Prazo estimado

## DADOS IMPORTANTES

### Banco de Dados Completo

O sistema tem 8 tabelas principais:

1. **profiles** - Usuários (admin, funcionários, clientes)
2. **projects** - Obras (4 obras do seu tio)
3. **tasks** - Tarefas específicas em cada obra
4. **check_ins** - Registro de presença dos funcionários
5. **photos** - Fotos de progresso e evidências
6. **issues** - Problemas e alertas
7. **comments** - Comentários em tarefas/projetos
8. **notifications** - Notificações do sistema

### Segurança

- Sistema de permissões (RLS) garante que:
  - Admin vê TUDO
  - Funcionários veem só suas tarefas
  - Clientes veem só suas obras
- Ninguém pode ver dados de outros

## FLUXO DE TRABALHO TÍPICO

### Manhã:
1. Funcionário chega na obra
2. Faz check-in no app
3. Vê suas tarefas do dia
4. Começa a trabalhar

### Durante o Dia:
5. Tira fotos do progresso
6. Marca tarefas como concluídas
7. Se tem problema, reporta no sistema
8. Admin recebe notificação

### Admin (de qualquer lugar):
9. Abre o dashboard
10. Vê quem está trabalhando
11. Vê tarefas aguardando aprovação
12. Aprova ou rejeita com notas
13. Se tem problema, vê e resolve

### Final do Dia:
14. Funcionário faz check-out
15. Sistema calcula horas trabalhadas
16. Cliente recebe atualização automática
17. Admin vê resumo do dia
